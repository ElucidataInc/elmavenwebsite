El-Maven

An open-source mass spectrometry data processing engine.

Why El-MAVEN?

El-MAVEN is an open-source mass spectrometry data processing engine that is optimal for isotopomer labeling and global metabolomic profiling experiments. It is designed to reduce complexity in metabolomics analysis by using an intuitive interface for exploring experimental data. The tool aims to make research faster than ever before.

Fast

Processes large datasets in seconds to display peaks.

Robust

El-MAVEN is more robust than MAVEN.

User Friendly

El-MAVEN is more intuitive compared to MAVEN.

Visualization Reimagined

Track a metabolite across samples with a single visual. Visualize all peaks originating from the same ion as a single group.
